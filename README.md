# SPIN-JEvo
SPIN-JEvo: a few-shot virtual directed evolution framework that couples LoRA-tuned ESM-2 with a genetic algorithm to rapidly explore protein sequence space and discover high-activity remote homologs from small binary-labeled sequence sets.

The LoRA adapter weights fine-tuned on TadA and CcdA are available at https://doi.org/10.6084/m9.figshare.31101862
